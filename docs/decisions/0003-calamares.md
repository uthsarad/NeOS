# 3. Use Calamares as the GUI installer

Date: 2026-02-17
Status: Accepted

## Context
We need a user-friendly, customizable, and reliable graphical installer for NeOS on x86_64 architectures.

## Decision
We will use the Calamares framework for the graphical installer on x86_64.

## Consequences
- Familiar and accessible installation experience.
- Highly customizable via modules.
- Currently limits the GUI installer availability to x86_64.
