## 🌟 What is NeOS?

**NeOS** is a curated, snapshot-based Arch Linux desktop distribution focused on predictable behavior, low breakage, and a Windows-familiar **KDE Plasma 6** experience. It is not a thin Arch wrapper or a DIY remix; it is a meticulously engineered sanctuary for users who demand stability without sacrificing the cutting-edge.

### ✨ Key Features
- **🚀 Snapshot-Based Rolling Release**: Coherent, tested package sets promoted via validated snapshots.
- **🎨 Elite UX**: A polished KDE Plasma 6 environment tailored for Windows switchers.
- **🛡️ Hardened Security**: Integrated UFW, systemd sandboxing, and curated security configurations.
- **⚡ Performance First**: Powered by the **Linux Zen** kernel, ZRAM compression, and Btrfs snapshots.
- **📦 Reliable Installation**: Streamlined deployment via a branded **Calamares** installer.

## 🛠️ Building the Sanctuary

To build the NeOS ISO locally, ensure you have `archiso` installed and run the following command from the root of the repository:

```bash
sudo ./build.sh
```

> **Note:** The ISO is automatically built and released to the **[GitHub Releases](https://github.com/uthsarad/NeOS/releases)** section upon every push to `main`, provided all validation tests pass.
